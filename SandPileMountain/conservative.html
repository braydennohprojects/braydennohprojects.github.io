Conservative Sandpile

A sandpile where toppling depends on the relative height to Moore neighbours.

Rule: find the steepest neighbour. If Δz ≥ threshold, transfer give blocks to that neighbour.

D4: 4 cardinal neighbours.
D8: all 8 neighbours (Moore).
Voronoi: ~1000 random cells, no directional bias.

Material is conserved. Edge cells drain when height ≥ threshold but no neighbour qualifies. By Brayden Noh.

Toppling Rule

For each cell i with height zi, find the neighbour j* with the largest height difference:
Δz = zi − zj* = maxj ∈ neighbours(zi − zj)

If Δz ≥ threshold:
  zi ← zi − give
  zj* ← zj* + give

Repeat until no cell can topple (relaxation).
Default: threshold = 4, give = 2

Interactive Demo

Click any cell to drop a block (+1). When Δz ≥ 4 to a neighbour, the cell topples (loses 2, steepest neighbour gains 2). Cascades animate step by step. Uses D8 (all 8 neighbours).
