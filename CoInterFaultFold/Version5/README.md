# Version 5

This page will explore how to incorporate long-term uplift in subduction zone modeling. In reality, it is an ideal extension of the ramp-flat geometry of the previous versions but the idea is that the uplift in previous ideas were all thought as overthrust, because the footwall is considered to be fixed in some reference frame. For subduction zones, this is not the case as the footwall is subducting, therefore the uplift may all be subducted producing 0 uplift in the hanging wall. In fact, this is the idealized model in Savage's original paper as he assumed 0 surface deformation. The current iteration of thought experiment and code beautifully extends both subduction zones (0 uplift) and continental zones (non-zero uplift) using a single parameter called alpha which controls the partition of overthrust and underthrust. Because interseismic is just the difference between the steady state and coseismic, the single model works for all colliding plates.
