How does interseismic and long-term uplift rate relate to each other?

This is a thought experiment to incorporate long-term uplift in subduction plate. In reality, it is a simple extension of the ramp-flat geometry but uplift in previous models were all thought as overthrust, because the footwall is considered to be fixed in some reference frame. For subduction zones, this is not the case as the footwall is subducting, therefore the uplift may all be subducted producing 0 uplift in the hanging wall. In fact, this is the idealized model in Savage's original paper as he assumed 0 surface deformation. The current iteration of thought experiment and code extends both subduction zones (0 uplift) and continental zones (non-zero uplift) using a single parameter called alpha which controls the partition of overthrust and underthrust. At the observer triangle, the relative velocity is constant regardless of alpha, but the partition controls how much of that convergence is accommodated by the footwall moving versus the hanging wall moving. What is currently unclear is what happens at the mantle wedge, often called the cold nose, and perhaps there are additional phenomena which the current model lacks to capture.
